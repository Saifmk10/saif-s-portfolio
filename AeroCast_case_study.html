AEROCAST

AeroCast is an app that transforms the way we experience weather . With its friendly commentary based on weather and user-friendly interface, AeroCast brings the excitement of live performances right to your fingertips

PROBLEM SOLVED

AeroCast is a real-time weather platform designed to keep users consistently informed with accurate and up-to-date weather conditions. It addresses the common challenge of limited access to live weather updates and personalized forecasts by delivering timely, location-specific information through a clean and intuitive user interface. Unlike traditional weather apps, AeroCast allows users to search for weather data from any location across the globe, providing precise and reliable forecasts that enhance situational awareness and improve planning.

DESIGN AND IMPLEMENTATION

Development of AeroCast took over a month to write the code and to learn the tech stacks involved in building the application. This application is build on flutter completely and as of now is only available on Android. Some of the widgets within the app are using some widgets such as : Geolocator , Rest API , OpenWeatherBit API. Logics to build the app were written by own and completely understandable to fellow developers in the future developement, The project has be well documented and the file organization is clear and concise.
